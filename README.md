#  Практическая работа №8 — IPC с использованием каналов

**Тема:** Межпроцессное взаимодействие (Inter-Process Communication) средствами языка C.  
**Цель:** Изучение работы неименованных каналов (`pipe`) и системных вызовов POSIX для организации обмена данными между процессами.

---

## Содержание

В репозитории представлены программные примеры, реализующие передачу данных между процессами с помощью системных вызовов `pipe()`, `fork()`, `dup2()`, `read()`, `write()` и `exec()`.

| Файл | Назначение |
|------|-------------|
| `pipe_ls_wc.c` | Реализация конвейера `ls | wc -l` — подсчёт количества файлов и каталогов. |
| `pipe_who_wc.c` | Реализация конвейера `who | wc -l` — подсчёт активных пользователей системы. |
| `pipe_oneway.c` | Односторонняя передача данных от родительского процесса к дочернему. |
| `pipe_twoway.c` | Двусторонний обмен сообщениями между родительским и дочерним процессами. |

---

##Компиляция и запуск

```bash
gcc pipe_ls_wc.c -o pipe_ls_wc
./pipe_ls_wc

gcc pipe_who_wc.c -o pipe_who_wc
./pipe_who_wc

gcc pipe_oneway.c -o pipe_oneway
./pipe_oneway

gcc pipe_twoway.c -o pipe_twoway
./pipe_twoway
Описание

Программы демонстрируют работу неименованных каналов и передачу данных между процессами без участия оболочки.
На практике реализованы примеры как одностороннего обмена, так и полноценного двустороннего взаимодействия.
